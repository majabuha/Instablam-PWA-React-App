INSTABLAM

Exam application using React js, subject Advanced JavaScript.

Deployed at: https://instablam-1.surge.sh/

PWA Camera app using geolocation and media devices,
with functionality of turning camera on and off,
taking picture, looking picture in the gallery, displaying
time and location where it was taken, downloading and deleting taken 
pictures. Given period: one week.

To start the app run --npm install-- in folder terminal, followed by 
--npm start-- which should automatically open app in your browser 
of choice on http://localhost:3000/.
